/**
    H7oscillo-v1 Readme
  
	Description:
	
	1) Full Low Level implementation.
	2) Displays two input waveforms on the LCD. Ch1 is Arduino-connector A0, Ch2 is Arduino-connector A1. 
	   Quadrature encoder is used to change the timescale. LCD refresh rate is 59.3Hz.
	3) Touch screen is implemented with polling every 20ms.
	4) uGUI is integrated. Two display layers are used - Foreground layer is for uGUI, background for waveforms.
	5) A minimizable MENU window is added, with 4 buttons for horizontal and vertical scale inc/dec.
